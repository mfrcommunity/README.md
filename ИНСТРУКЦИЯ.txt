1. Распаковать файлы в удобное вам место.
2. Запуститьf4Njsl.exe [можете его переименовать, если вам хочется]
2. Дождаться загрузки.
3. После запуска у вас появится окно с кодом, его нужно предоставить Даниилу Мелихову.
4. После предоставления кода нужно дождаться активации.
5. После активации у вас появится окно с запуском скрипта, нажмите ок с активным окном PUBG.
6. Бинды работают только в матче, не пытайтесь их активировать в меню.


Функции:

F1 Быстрое прицеливание [Включение/Выключение]
F2 Автоматическая стрельба [Включение/Выключение]
F3 Уменьшение отдачи [Включение/Выключение] По умолчанию стоит 5 (для прицелев коллиматорного, голографического)
F4 Дальний прыжок
F7 Прибавление отдачи [-5]
F8 Убавление отдачи [+5]
Numpad - Прибавление отдачи [-1]
Numpad + Убавление отдачи [+1]
15+ и более для прицелов 4x 8x 12x 15x
На клавишу 9 ваш персонаж автоматически использует 6 бинтов подряд без заддержки.
На клавишу 6 ваш персонаж автоматически использует 4 энергетика без задежрки.
На клавишу 0 ваш персонаж автоматически использует 2 болеутоляющего без задержки.
На клавишу = ваш персонаж начинает автоматически бежать (приостановить можно нажатием клавишей w,a,s,d или же на = ) также при применение  чего-нибудь автоматически отключается. К примеру открыть дверь.
Автолутание активируется на Ё (`). Чтобы его настроить нужно выполнить следующее.. 
Сделать скриншот экрана, открыть паинт.. вставить скриншот и навести курсор мыши.. (ниже написано)
Окно для ведения 1 - X
Окно для ведения 2 - Y
-— ОТКУДА БУДЕТ БРАТЬСЯ ЛУТ
Окно для ведения 3 - X2
Окно для ведения 4 - Y2
-— ПРИМЕРНО СЕРЕДИНА ЭКРАНА

F6 Закрыть скрипты [РАБОТАЕТ ТОЛЬКО В МАТЧЕ]
